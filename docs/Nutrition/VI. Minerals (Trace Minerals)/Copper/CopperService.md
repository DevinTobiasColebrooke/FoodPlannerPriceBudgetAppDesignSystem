# Copper Service

## DRI Type
- EAR and RDA established

## Criteria for EAR
- Combination of biochemical indicators:
  - Plasma copper
  - Serum ceruloplasmin
  - Platelet copper concentrations
  - Erythrocyte superoxide dismutase activity

## Values
- EAR, RDA (µg/d) provided
  - Adult RDA: 900 µg/day

## UL
- 10,000 µg/d for adults
