13. Alerts & Notifications (`.alert-fp`)

Alerts provide contextual feedback messages for user actions or system events.

Base Alert Style

  • Padding: p-4
  • Border Radius: rounded-md (or rounded-fp)
  • Layout: Flexbox to align icon and text (flex items-start space-x-3)
  • Icon: Relevant status icon (SVG)
  • Text: Clear, concise message, text-sm.
  • Coloring: Use semantic colors for background and text.

Alert Types

Success Alert

Profile Updated!

Your changes have been saved successfully.

Error/Alert

Action Failed!

There was an issue processing your request. Please try again.

Warning Alert

Budget Exceeded

This meal plan is slightly over your set budget. Adjust plan?

Info Alert

Tip of the Day

Drinking water before meals can help with portion control!

Alert Usage Examples

  • "Download my Data" prompt (can be a more prominent banner or a dismissible alert).
  • Form submission success/error messages.
  • Validation errors within forms (can be inline or a summary alert).
  • General app notifications, tips, or system status updates.
  • Budget warnings or suggestions.

Alerts can be dismissible by adding a close button.
