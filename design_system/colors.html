3.4. Semantic Colors

Success: This is a success message. (bg-fp-success-bg text-fp-success)
Error/Alert: This is an error message. (bg-fp-error-bg text-fp-error)
Warning: This is a warning message. (bg-fp-warning-bg text-fp-warning)
Info: This is an informational message. (bg-fp-info-bg text-fp-info)

Focus Ring: Use focus:ring-fp-primary-green or focus:ring-fp-accent-blue.
